# UrbanFinder

**UrbanFinder** is a comprehensive property rental application that simplifies the process of finding, managing, and booking properties. It provides users with an intuitive interface and real-time property information.

## Features

- Browse and search properties by location, price, and type.
- Detailed property listings with images, descriptions, and amenities.
- Favorites and bookmarks for quick access to preferred properties.
- Contact landlords or property managers directly through the app.
- User authentication for secure and personalized experience.
- Responsive and interactive UI for seamless navigation.

  ## Tech Stack

- **Frontend:** Jetpack Compose, Kotlin, NodeJs
- **Backend:** Firebase Firestore, Firebase Authentication
- **Storage:** Firebase Storage for property imagesm Img
- **State Management:** Provider
- **Maps Integration:** OSM Maps
